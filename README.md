
# ArtNetConsole

DMX Lighting Software for smal events.
Using ArtNet as outgoing protocoll.

## Features
- Patch
    - Individual Fixtures
- Scenes
    - save and read
- Midi control
- Art Net
    - 1 Univers (DMX 512)
- Java
    - runs on any system
    - new java 19
    - Terminal based
